# 🩺 Diabetes Prediction Mini-Project

A Machine Learning web application that predicts the likelihood of diabetes based on health metrics using a Random Forest Classifier.

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/ABIZER-web/Diabetes-Prediction.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python -m streamlit run app.py`

## 📊 Dataset
Used the Pima Indians Diabetes Dataset from Kaggle.
